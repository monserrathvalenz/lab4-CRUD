# Lab 4 - CRUD Simple

Aplicación web desarrollada con React que implementa las operaciones básicas 
CRUD (Create, Read, Update, Delete) sobre un catálogo de datos.

## Tecnologías utilizadas
- React
- Bootstrap
- Reactstrap

## Funcionalidades
- **Crear** nuevos registros mediante un modal
- **Leer** los datos en una tabla
- **Editar** registros existentes
- **Eliminar** registros con confirmación
